# AI Slop Detector

> A lightweight Flask app that estimates whether text was written by a human or generated by AI.

AI Slop Detector combines a RoBERTa-based detector with lightweight phrase heuristics to score pasted text or scraped article content from a URL. It is built for quick, simple checks in a clean browser UI.

## Why This Project?

- Fast local workflow with a minimal Flask setup
- Dual-signal scoring instead of a single detector output
- Supports both direct text input and URL scraping
- Breaks long text into chunks for more stable analysis
- Clean UI with light/dark mode support

## How It Works

The app blends two signals:

- `Hello-SimpleAI/chatgpt-detector-roberta` via Hugging Face Transformers for primary AI-vs-human classification
- A heuristic phrase scanner that looks for common AI-style patterns and overused wording

The model score is weighted more heavily, and the heuristic score provides a smaller adjustment.

## Features

- Paste text and analyze it instantly
- Scrape a public article URL and analyze the extracted content
- View AI score, human score, confidence, and chunking metadata
- Handle longer documents by splitting them into smaller chunks
- Open an in-app info modal for tool explanation and limitations

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/mijnap1/AI-SLOP-Detector.git
cd AI-SLOP-Detector
```

### 2. Create and activate a virtual environment

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the app

```bash
python app.py
```

Then open [http://127.0.0.1:5000](http://127.0.0.1:5000).

## Notes

- The first run may take longer because the Hugging Face model has to download.
- URL scraping works best on article-style pages with readable main content.
- Detection accuracy drops on very short text or heavily edited AI output.
- Results should be treated as a signal, not proof.
